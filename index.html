Padding

Tag / Syntax Name/Type Description
Padding is a space inside the elements
padding-top: Syntax use to assing padding on top of the elements
padding-right: Syntax use to assing padding on the right of the elements
padding-bottom: Syntax use to assing padding at the bottom of the elements
padding-left: Syntax use to assing padding on the left of the elements

Margin

Tag / Syntax Name/Type Description
Margin is a space outside the elements
margin-top: Syntax use to assing margin on top of the elements
margin-right: Syntax use to assing margin on the right of the elements
margin-bottom: Syntax use to assing margin at the bottom of the elements
margin-left: Syntax use to assing margin on the left of the elements
margin: 0 auto; Shorthand Syntax First number is value of top and bottom margin, the second is for sides and auto mean set it default
